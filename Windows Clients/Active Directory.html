Was ist ein Active Directory?

Das Active Directory (AD) ist eine Verzeichnisdienst-Software von Microsoft, die zur Verwaltung von Benutzerinformationen, Ressourcen im Netzwerk und Sicherheitsrichtlinien verwendet wird.

Hauptfunktionen von Active Directory
- Zentralisierte Verwaltung von Benutzerkonten und Gruppen
- Bereitstellung von Zugriffsrechten auf Ressourcen im Netzwerk
- Durchsetzung von Sicherheitsrichtlinien
- Unterstützung für Single Sign-On (SSO) und Authentifizierungsdienste

Verwendung von Active Directory in einem Netzwerk

Active Directory wird in Unternehmen und Organisationen eingesetzt, um die Verwaltung und Sicherheit von Netzwerken zu verbessern. Es dient als zentraler Ort zur Speicherung und Verwaltung von Netzwerkressourcen und Benutzerdaten.

Domain Controller

Ein Domain Controller (DC) ist ein Server, der das Active Directory verwaltet und die Authentifizierung von Benutzern und Computern in einer Windows-Domäne ermöglicht.

Rolle der Active Directory Datenbank

Die Active Directory Datenbank speichert alle Informationen über Objekte in der Domäne, einschließlich Benutzer, Gruppen, Computer und Ressourcen. Sie ist entscheidend für die Funktionalität und Sicherheit der Domäne.

Globaler Katalog (Global Catalog)

Der Globale Katalog ist eine spezielle Instanz eines Domain Controllers, die eine Teilmengen der Active Directory Datenbank enthält. Er dient dazu, schnelle Suchanfragen über mehrere Domänen hinweg zu ermöglichen und ist entscheidend für die Anmeldung von Benutzern in einer Gesamtstruktur.

Aufgaben des AD Replication Service

Der AD Replication Service ist verantwortlich für die Replikation von Daten zwischen verschiedenen Domain Controllern in einem Active Directory Netzwerk, um sicherzustellen, dass alle Domänencontroller synchronisierte Kopien der Active Directory Datenbank halten.

Domains, Trees und Forests

Begriff Beschreibung
Domain Eine Domäne ist eine logische Gruppe von Netzwerkobjekten (Benutzer, Computer), die eine Sicherheitsgrenze darstellt und eine einheitliche Benutzeranmeldung ermöglicht.
Tree Ein Tree (Baum) besteht aus mehreren Domänen, die hierarchisch miteinander verbunden sind und sich eine gemeinsame DNS-Namensstruktur teilen.
Forest Ein Forest (Gesamtstruktur) ist eine Sammlung von Trees, die durch eine gemeinsame Schema- und Konfigurationspartition in einer Active Directory Umgebung verbunden sind.

Unterschied zwischen organisational Units (OUs) und Container

Eigenschaft Organisational Units (OUs) Container
Verwendungszweck Für die Organisation von Objekten und Anwendung von Gruppenrichtlinien verwendet. Dienen primär zur Verwaltung und Organisation von Objekten.
Delegierte Verwaltung Können Sicherheitsrichtlinien delegieren und anwenden. Können keine Sicherheitsrichtlinien delegieren.
Benennung Haben eigene Namen und sind leicht identifizierbar. Werden durch ihre Distinguished Names (DN) identifiziert.

Arten von Objekten in Active Directory

In Active Directory werden verschiedene Arten von Objekten verwaltet, darunter:

  • Benutzer
  • Computer
  • Gruppen (z.B. Sicherheitsgruppen, Verteilergruppen)
  • Drucker
  • Dateifreigaben
  • Kontakte

Group Policy

Eine Group Policy (Gruppenrichtlinie) definiert Regeln und Einstellungen für Benutzer und Computer in einer Domäne. Sie ermöglicht die zentrale Verwaltung und Konfiguration von Sicherheits- und Desktopeinstellungen.

Zentrale Verwaltung von Benutzern und Gruppenrichtlinien

Benutzer und Gruppenrichtlinien können zentral über den Group Policy Management Editor (GPMC) verwaltet werden. Administratoren können Richtlinien erstellen, anwenden und verwalten, die dann auf Benutzer und Computer in der gesamten Domäne angewendet werden.

Vorteile von Active Directory gegenüber anderen Verzeichnisdiensten

Vorteile
- Integration mit Windows-Betriebssystemen und Diensten
- Zentrale Verwaltung von Netzwerkressourcen und Sicherheitseinstellungen
- Skalierbarkeit für große Netzwerke
- Unterstützung für Single Sign-On (SSO) und sichere Authentifizierung
