Die Datenträgerverwaltung in Windows ermöglicht die Verwaltung von Festplatten und Partitionen. Hier ist eine Schritt-für-Schritt-Anleitung für beide Versionen:

In Windows 10:

  1. Öffnen der Datenträgerverwaltung:

    • Klicken Sie mit der rechten Maustaste auf das Startmenü und wählen Sie “Datenträgerverwaltung” aus.

  2. Partition erstellen/verwalten:

    • Klicken Sie mit der rechten Maustaste auf eine Festplatte oder einen nicht zugewiesenen Bereich und wählen Sie eine Aktion wie “Neues einfaches Volume erstellen” oder “Volume verkleinern”.

  3. Formatieren von Partitionen:

    • Wählen Sie eine Partition aus und klicken Sie mit der rechten Maustaste darauf, dann wählen Sie “Volume formatieren”.

  4. Festplatten verwalten:

    • Rechtsklick auf eine Festplatte ermöglicht Optionen wie “Laufwerkbuchstaben und Pfade ändern” oder “Festplatte offline/online”.

In Windows 11:

  1. Öffnen der Datenträgerverwaltung:

    • Suchen Sie nach “Datenträgerverwaltung” im Startmenü und wählen Sie die entsprechende Option aus.

  2. Partition erstellen/verwalten:

    • Verwenden Sie die gleichen Schritte wie in Windows 10, um Volumes zu erstellen oder zu verwalten.

  3. Formatieren von Partitionen:

    • Die Formatierungsoptionen sind ähnlich wie in Windows 10 durch Rechtsklick auf die Partition und Auswahl von “Volume formatieren”.

  4. Festplatten verwalten:

    • Die Verwaltung von Festplatten durch Rechtsklick bietet ähnliche Optionen wie in Windows 10.

Diese Schritte ermöglichen eine einfache Verwaltung von Festplatten und Partitionen in beiden Windows-Versionen.
