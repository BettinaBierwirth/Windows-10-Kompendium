Einrichtung eines virtuellen Windows 10 in Hyper-V

  1. Öffnen von Hyper-V Manager:

    • Starten Sie den Hyper-V Manager auf Ihrem Windows-PC.

  2. Neue virtuelle Maschine erstellen:

    • Klicken Sie auf “Neue virtuelle Maschine erstellen” im rechten Bereich des Hyper-V Managers.

  3. Konfigurieren der virtuellen Maschine:

    • Folgen Sie den Anweisungen im Assistenten zur Erstellung einer neuen virtuellen Maschine. Legen Sie fest:
      • Namen und Speicherort der virtuellen Maschine
      • Generation der virtuellen Maschine (empfohlen: Generation 2 für Windows 10)
      • Zuweisung von Arbeitsspeicher und Netzwerkkonfiguration

  4. Installation von Windows 10:

    • Montieren Sie das Windows 10 ISO-Image und starten Sie die virtuelle Maschine.
    • Folgen Sie den Installationsanweisungen von Windows 10.

  5. Einrichtung von Windows 10:

    • Führen Sie die grundlegende Einrichtung von Windows 10 durch, wie die Konfiguration von Benutzerkonten und Netzwerkeinstellungen.

Einrichtung eines neuen Nutzers für Remote-Zugriff

  1. Erstellen eines neuen Benutzers:

    • Gehen Sie zu “Einstellungen” > “Konten” > “Familie & weitere Benutzer”.
    • Klicken Sie auf “Jemand anderem zu diesem PC hinzufügen”.
    • Fügen Sie die erforderlichen Informationen für den neuen Benutzer hinzu.

  2. Zugriffsrechte des neuen Nutzers:

    • Stellen Sie sicher, dass der neue Benutzer über Administratorrechte verfügt, um Remote-Zugriff durchführen zu können.
    • Sie können dies über “Einstellungen” > “Konten” > “Ihr Konto” > “Administratoren” prüfen und anpassen.

Firewall-Einstellungen für Remote-Zugriff

  1. Öffnen der Windows Defender Firewall:

    • Suchen Sie nach “Windows Defender Firewall mit erweiterter Sicherheit” und öffnen Sie es.

  2. Eingehende Regeln erstellen:

    • Klicken Sie auf “Eingehende Regeln” und dann auf “Neue Regel erstellen”.
    • Wählen Sie “Port” aus und klicken Sie auf “Weiter”.
    • Wählen Sie “TCP” und geben Sie den spezifischen Port ein, der für Remote-Zugriff (z.B. Port 3389 für RDP) verwendet wird. Klicken Sie auf “Weiter”.
    • Wählen Sie “Zulassen die Verbindung” und klicken Sie auf “Weiter”.
    • Geben Sie einen Namen und eine optionale Beschreibung für die Regel ein und klicken Sie auf “Fertig stellen”.

  3. Netzwerkerkennung anpassen:

    • Gehen Sie zu “Einstellungen” > “Netzwerk und Internet” > “Ethernet” (oder “WLAN”) > “Netzwerkerkennung”.
    • Stellen Sie sicher, dass das Netzwerk als “Privat” eingestuft ist, um Remote-Zugriff zuzulassen.

Zusammenfassung

Nach der Einrichtung eines virtuellen Windows 10 in Hyper-V und der Erstellung eines neuen Benutzers mit Administratorrechten müssen Sie entsprechende Firewall-Regeln erstellen, um Remote-Zugriff zuzulassen. Vergewissern Sie sich, dass das Netzwerk als “Privat” eingestuft ist, um sicherzustellen, dass der Remote-Zugriff ordnungsgemäß funktioniert.
