Upgrade-Pfade zu Windows 10

Frühere Windows-Version Unterstützte Editionen Ziel-Editionen Windows 10 Bemerkungen
Windows 7 Starter Alle Windows 10 Home Direktes Upgrade möglich
Windows 7 Home Basic Alle Windows 10 Home Direktes Upgrade möglich
Windows 7 Home Premium Alle Windows 10 Home Direktes Upgrade möglich
Windows 7 Professional Alle Windows 10 Pro Direktes Upgrade möglich
Windows 7 Ultimate Alle Windows 10 Pro Direktes Upgrade möglich
Windows 8.1 Alle Windows 10 Home/Pro Abhängig von der Edition
Windows 8.1 Pro Alle Windows 10 Pro Direktes Upgrade möglich
Windows 8.1 Enterprise Alle Windows 10 Enterprise Volume Licensing erforderlich
Windows 8.1 RT Keine Keine Kein Upgrade-Pfad verfügbar
Windows 8 Alle Windows 10 Home/Pro Zuerst Update auf Windows 8.1 erforderlich
Windows 7 Enterprise Volume Licensing Windows 10 Enterprise Volume Licensing erforderlich

Upgrade-Pfade zu Windows 11

Frühere Windows-Version Unterstützte Editionen Ziel-Editionen Windows 11 Bemerkungen
Windows 10 Home Alle Windows 11 Home Direktes Upgrade möglich
Windows 10 Pro Alle Windows 11 Pro Direktes Upgrade möglich
Windows 10 Enterprise Volume Licensing Windows 11 Enterprise Volume Licensing erforderlich
Windows 10 Education Alle Windows 11 Education Direktes Upgrade möglich
Windows 8.1 Alle Keine Zuerst Upgrade auf Windows 10 erforderlich
Windows 7 Alle Keine Zuerst Upgrade auf Windows 10 erforderlich
