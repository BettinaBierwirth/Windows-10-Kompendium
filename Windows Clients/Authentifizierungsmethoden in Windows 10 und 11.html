Accounttypen

Accounttyp Beschreibung Nutzung Vorteile Nachteile
Lokales Konto Ein Benutzerkonto, das nur auf einem einzelnen Computer existiert. Einzelbenutzerumgebungen, Computer ohne Internetverbindung Unabhängig von Internetverbindung Keine Synchronisierung mit anderen Geräten
Microsoft-Konto Ein Online-Konto, das mehrere Geräte und Dienste synchronisiert. Geräteübergreifende Nutzung, Cloud-Dienste Synchronisation von Einstellungen und Daten Abhängigkeit von Microsoft-Diensten
Domänenkonto Ein Konto, das von einer Organisation über Active Directory verwaltet wird. Unternehmensumgebungen, Netzwerke Zentrale Verwaltung, erhöhte Sicherheit Erfordert Serverinfrastruktur

Windows Hello

Windows Hello bietet biometrische Authentifizierungsmethoden wie Gesichtserkennung, Fingerabdruck und PIN. Diese Methode erhöht die Sicherheit und Benutzerfreundlichkeit.

Funktionsweise von Windows Hello

  • Gesichtserkennung: Verwendet die Kamera, um das Gesicht des Benutzers zu erkennen und zu authentifizieren.
  • Fingerabdruck: Verwendet einen Fingerabdruckscanner, um den Benutzer zu authentifizieren.
  • PIN: Eine numerische PIN, die lokal auf dem Gerät gespeichert wird.

Konfiguration von Windows Hello

  1. Gesichtserkennung einrichten

    1. Öffnen Sie Einstellungen.
    2. Gehen Sie zu Konten > Anmeldeoptionen.
    3. Wählen Sie Windows Hello Gesichtserkennung und klicken Sie auf Einrichten.
    4. Folgen Sie den Anweisungen, um Ihr Gesicht zu registrieren.

  2. Fingerabdruck einrichten

    1. Öffnen Sie Einstellungen.
    2. Gehen Sie zu Konten > Anmeldeoptionen.
    3. Wählen Sie Windows Hello Fingerabdruck und klicken Sie auf Einrichten.
    4. Folgen Sie den Anweisungen, um Ihren Fingerabdruck zu registrieren.

  3. PIN einrichten

    1. Öffnen Sie Einstellungen.
    2. Gehen Sie zu Konten > Anmeldeoptionen.
    3. Wählen Sie PIN und klicken Sie auf Hinzufügen.
    4. Folgen Sie den Anweisungen, um eine PIN zu erstellen.

Benutzerkontensteuerung (UAC)

Die Benutzerkontensteuerung (User Account Control, UAC) ist ein Sicherheitsfeature, das verhindert, dass unbefugte Änderungen an Ihrem System vorgenommen werden. UAC fordert den Benutzer zur Bestätigung auf, wenn administrative Berechtigungen erforderlich sind.

Funktionsweise von UAC

  • Standardbenutzer: Erhalten eine Aufforderung zur Eingabe eines Administrator-Kennworts, wenn administrative Aufgaben ausgeführt werden.
  • Administratoren: Erhalten eine Bestätigungsaufforderung zur Durchführung administrativer Aufgaben.

Konfiguration von UAC

  1. Öffnen Sie Systemsteuerung.
  2. Navigieren Sie zu Benutzerkonten > Benutzerkonten > Einstellungen der Benutzerkontensteuerung ändern.
  3. Stellen Sie den Schieberegler auf das gewünschte Benachrichtigungsniveau.

Unterschiede zwischen Windows 10 und Windows 11

Feature Windows 10 Windows 11
Microsoft-Konto Optionale Anmeldung Stärkere Integration und bevorzugte Anmeldung
Windows Hello Basis-Biometrie-Unterstützung Erweiterte Biometrie-Optionen, verbesserte Leistung
Benutzerkontensteuerung (UAC) Standardmäßige Sicherheitsabfragen Verbesserte Sicherheitsabfragen und -benachrichtigungen

Zusammenfassung

In Windows 10 und 11 gibt es mehrere Authentifizierungsmethoden, die auf verschiedene Bedürfnisse und Sicherheitsanforderungen zugeschnitten sind. Lokale Konten bieten Unabhängigkeit, während Microsoft-Konten und Domänenkonten erweiterte Synchronisierungs- und Verwaltungsfunktionen bieten. Windows Hello bietet eine bequeme und sichere biometrische Authentifizierung, während UAC dazu beiträgt, unbefugte Änderungen am System zu verhindern.

Windows Hello Fingerabdruck Einstellungen > Konten > Anmeldeoptionen > Windows Hello Fingerabdruck > Einrichten
Windows Hello PIN Einstellungen > Konten > Anmeldeoptionen > PIN > Hinzufügen
