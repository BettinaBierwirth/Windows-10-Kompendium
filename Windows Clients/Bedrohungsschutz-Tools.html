Schritt-für-Schritt-Anleitung zur Anwendung der Sicherheitsmaßnahmen

Windows Defender aktivieren

  1. Öffnen Sie die Einstellungen über das Startmenü.
  2. Navigieren Sie zu Update und Sicherheit.
  3. Wählen Sie Windows-Sicherheit und dann Viren- und Bedrohungsschutz.
  4. Stellen Sie sicher, dass der Echtzeitschutz aktiviert ist.

BitLocker aktivieren

  1. Öffnen Sie die Systemsteuerung.
  2. Navigieren Sie zu System und Sicherheit > BitLocker-Laufwerksverschlüsselung.
  3. Klicken Sie auf BitLocker aktivieren.
  4. Folgen Sie den Anweisungen zum Einrichten der Verschlüsselung.

Verschlüsselungsmethoden: EFS und BitLocker

EFS (Encrypting File System)

EFS ermöglicht die Verschlüsselung von Dateien und Ordnern auf NTFS-Dateisystemen. Dies schützt sensible Daten vor unbefugtem Zugriff.

Anleitung zur Konfiguration von EFS

  1. Rechtsklicken Sie auf die Datei oder den Ordner, den Sie verschlüsseln möchten.
  2. Wählen Sie Eigenschaften.
  3. Klicken Sie auf Erweitert.
  4. Aktivieren Sie die Option Inhalt verschlüsseln, um Daten zu schützen.
  5. Bestätigen Sie mit OK.

BitLocker

BitLocker verschlüsselt das gesamte Laufwerk und schützt so alle Daten auf dem Laufwerk vor unbefugtem Zugriff.

Anleitung zur Konfiguration von BitLocker

  1. Öffnen Sie die Systemsteuerung.
  2. Gehen Sie zu System und Sicherheit > BitLocker-Laufwerksverschlüsselung.
  3. Klicken Sie auf BitLocker aktivieren.
  4. Folgen Sie den Anweisungen zur Auswahl der Verschlüsselungsmethode und zur Sicherung des Wiederherstellungsschlüssels.

Vergleich von EFS und BitLocker

Eigenschaft EFS BitLocker
Verschlüsselungsziel Einzelne Dateien und Ordner Gesamtes Laufwerk
Verfügbarkeit NTFS-Dateisysteme Verfügbar in bestimmten Windows-Versionen
Verwaltung Individuell für jede Datei/jeden Ordner Zentral über das Laufwerkverwaltungstool
Sicherheit Schutz auf Dateiebene Schutz auf Laufwerkebene

AppLocker

AppLocker ist ein Feature, das Administratoren ermöglicht, welche Anwendungen und Dateien Benutzer ausführen dürfen. Dies erhöht die Sicherheit, indem unerwünschte Software blockiert wird.

Funktionsweise von AppLocker

  1. Definieren von Regeln basierend auf Dateipfade, Dateiversionen und Signaturen.
  2. Anwenden von Regeln über Gruppenrichtlinien.
  3. Erzwingen der Richtlinien auf den Benutzercomputern.

Anleitung zur Konfiguration von AppLocker

  1. Öffnen Sie den Gruppenrichtlinien-Editor (gpedit.msc).
  2. Navigieren Sie zu Computerkonfiguration > Windows-Einstellungen > Sicherheitseinstellungen > Anwendungssteuerungsrichtlinien > AppLocker.
  3. Erstellen Sie neue Regeln unter Ausführbare Regeln, Windows Installer Regeln, Skriptregeln und DLL-Regeln.
  4. Definieren Sie die gewünschten Einschränkungen und wenden Sie diese an.
