Allgemeines

Provisionierung bezeichnet den Prozess der Vorbereitung und Konfiguration eines IT-Systems, um es einsatzbereit zu machen. Es ist zudem möglich mittels Provisionierung weitere Komponenten eines Windows 10 oder 11 Systems zu installieren oder zu konfigurieren.

Provisionierung von Windows 10

Microsoft stellt einige Tools zur Provisionierung zur Verfügung.

Windows Imaging and Configuration Designer (Windows ICD):

Ein Tool innerhalb des Windows Assessment and Deployment Kit (ADK), das Administratoren hilft, Bereitstellungspakete zu erstellen.

Windows Autopilot:

Ein Cloud-basierter Dienst, der eine schnelle Bereitstellung und Konfiguration neuer Geräte ermöglicht. Geräte können direkt vom Hersteller an den Endbenutzer versendet werden, der nur noch die Anmeldung durchführen muss.

Microsoft Deployment Toolkit (MDT):

Ein kostenloses Tool von Microsoft, das für die Bereitstellung von Windows-Betriebssystemen und Anwendungen verwendet wird. MDT automatisiert viele Schritte des Bereitstellungsprozesses.

System Center Configuration Manager (SCCM):

Ein Verwaltungstool, das umfassende Lösungen für die Verwaltung von Windows-Computern bietet, einschließlich der Bereitstellung von Betriebssystemen, Software und Updates.

Erstellung eines Bereitstellungspakets:

Mithilfe von Windows ICD oder MDT wird ein Paket erstellt, das alle notwendigen Einstellungen und Anwendungen enthält.

Bereitstellung des Betriebssystems:

Das Bereitstellungspaket wird auf den Zielcomputern installiert, entweder über das Netzwerk (PXE-Boot), USB-Laufwerke oder direkt über den Hersteller.

Post-Bereitstellungskonfiguration:

Nach der Installation werden zusätzliche Konfigurationen vorgenommen, wie z.B. die Installation von Sicherheitsupdates und die Konfiguration von Gruppenrichtlinien.

Provisionierung von Windows 11

Wie auch bereits bei Windows 10 funktioniert die Provisionierung unter Windows 11 nach demselben Prinzip. Windows 11 hat aber eine erweiterte Unterstützung durch den Windows Autopilot, die sich durch eine bessere User-Experience auszeichnet, da einige neue Funktionen hinzugekommen sind die den Prozess vereinfachen. Neu ist die Integration mit Microsoft Endpoint Manager. Dies ermöglicht eine einheitliche Verwaltung von Geräten, Anwendungen und Daten.  Windows 11 führt zusätzliche Sicherheitsmaßnahmen ein, wie z.B. den obligatorischen Einsatz von TPM 2.0 (Trusted Platform Module) und erweiterte Schutzmechanismen gegen Malware und Ransomware.

Der Prozess ist ebenfalls ähnlich. Administratoren erstellen Bereitstellungspakete und konfigurieren Autopilot-Profile mithilfe von Windows ICD, MDT oder Endpoint Manager.  Neue Geräte können direkt an den Endbenutzer versendet werden. Nach dem Einschalten und der Anmeldung beim Unternehmensnetzwerk wird das Bereitstellungspaket automatisch angewendet. Windows 11 führt den Benutzer durch eine vereinfachte und intuitive Einrichtung, während im Hintergrund die notwendigen Anwendungen und Konfigurationen installiert werden. Nach der Bereitstellung werden Geräte kontinuierlich über Endpoint Manager oder SCCM verwaltet, um sicherzustellen, dass sie den aktuellen Sicherheits- und Konfigurationsrichtlinien entsprechen.
